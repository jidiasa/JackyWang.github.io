Project Description

This project aims to demonstrate various image processing techniques using Python. The core objectives include:

  • Applying Gaussian blur to images to smooth out noise.
  • Using different image alignment techniques for better image registration.
  • Implementing and testing various algorithms for image enhancement and colorization.

The project involves analyzing the effects of these techniques on different types of images and comparing the results. The provided Python scripts include implementations of these techniques, and the results are visualized in the form of processed images.
